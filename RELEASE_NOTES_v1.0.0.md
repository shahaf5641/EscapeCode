# EscapeCode Windows Build

This release contains the playable Windows version of EscapeCode.

## How To Play

1. Download `EscapeCode-Windows.zip`
2. Extract the archive
3. Open `EscapeCode.exe`

## Included In This Release

- Full playable Windows build
- Main menu, room progression, and ending flow
- Coding puzzle gameplay
- Eye tracking and microphone-click support where available

## Notes

- Unity is not required to play this build
- This public build does not support voice recording unless an OpenAI API key is configured by the developer

## Version

- Initial public Windows release
